# LGAG
Linux Gaming AGent

WARNING: THE AGENT TAKES A SCREENSHOT OF YOUR SCREEN EVERY 5 SECONDS!

WARNING: I TAKE NO RESPONSIBILITY FOR ANY HARM COMING TO YOUR SYSTEM OR WHATEVER ELSE!

Now that the important stuff is out let me describe how it works! 😄

1. You need to use the FISH shell for this to work. Garuda Linux or CachyOS are fine by default.
2. On the link provided there are two files. You need to run (or modify) run-agent.fish.
3. Install dependencies. Ask an LLM how to do that.
4. Run the game, any game on Steam is autodected or you type the exact name of the game when the script runs. This is pretty important so the agent answers properly and in relevance.
5. Run run-agent.fish in a terminal. Make sure the game you're about to play is detected/written correctly and start gaming! 😄 You talk to it and it replies within 20 seconds most usually. It runs on Ollama.
6. Repeat for other games! 😄

I have tested it in a couple of games like Fatekeeper and Cyberpunk 2077 and it seemed to mostly work. I mainly ask it about locations in the game and lore so take its gaming advice as you will. UwU

If anyone wants to fork the repo and do fixes/changes you're obviously welcome to do so! 😄
